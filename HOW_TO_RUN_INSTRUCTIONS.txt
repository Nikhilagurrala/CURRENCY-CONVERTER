STEP-BY-STEP INSTRUCTIONS TO RUN THE WEBSITE
============================================

METHOD 1: FROM GITHUB REPOSITORY (RECOMMENDED)
==============================================

1. INSTALL GIT (if not already installed)
   - Download Git from: https://git-scm.com/downloads
   - Install with default settings

2. CLONE THE REPOSITORY
   - Open Command Prompt or PowerShell
   - Run: git clone https://github.com/Nikhilagurrala/CURRENCY-CONVERTER.git
   - Navigate to folder: cd CURRENCY-CONVERTER

3. INSTALL PYTHON (if not already installed)
   - Download Python from: https://www.python.org/downloads/
   - Make sure to check "Add Python to PATH" during installation
   - Verify installation: python --version

4. INSTALL DEPENDENCIES
   - Run: pip install -r requirements.txt
   - Wait for all packages to install

5. RUN THE APPLICATION
   - Run: python app.py
   - Wait for "Running on http://127.0.0.1:5000" message

6. OPEN THE WEBSITE
   - Open your web browser
   - Go to: http://localhost:5000
   - OR: http://127.0.0.1:5000

METHOD 2: DIRECT DOWNLOAD FROM GITHUB
=====================================

1. GO TO GITHUB REPOSITORY
   - Visit: https://github.com/Nikhilagurrala/CURRENCY-CONVERTER

2. DOWNLOAD ZIP FILE
   - Click the green "Code" button
   - Select "Download ZIP"
   - Extract the ZIP file to your desired location

3. FOLLOW STEPS 3-6 FROM METHOD 1
   - Install Python
   - Install dependencies
   - Run the application
   - Open in browser

TROUBLESHOOTING COMMON ISSUES
============================

ISSUE: "python is not recognized"
SOLUTION: 
- Make sure Python is installed and added to PATH
- Try using "python3" instead of "python"
- Restart Command Prompt after Python installation

ISSUE: "pip is not recognized"
SOLUTION:
- Install pip: python -m ensurepip --upgrade
- Or use: python -m pip install -r requirements.txt

ISSUE: "Port 5000 is already in use"
SOLUTION:
- Stop other applications using port 5000
- Or change port: set FLASK_RUN_PORT=5001 && python app.py
- Then access: http://localhost:5001

ISSUE: "Module not found" errors
SOLUTION:
- Make sure you're in the correct directory (CURRENCY-CONVERTER folder)
- Run: pip install -r requirements.txt again
- Check if all dependencies installed successfully

ISSUE: Website not loading
SOLUTION:
- Check if the application is running (should show "Running on..." message)
- Try different URLs: http://localhost:5000, http://127.0.0.1:5000
- Check firewall settings
- Try incognito/private browsing mode
